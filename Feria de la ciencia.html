ESP32

Microcontrolador con Wi-Fi y Bluetooth para IoT.

ESP32

Microcontrolador avanzado con conectividad Wi-Fi y Bluetooth, ideal para automatización y control inteligente de dispositivos.

ESP8266

Placa con Wi-Fi integrada para proyectos de domótica.

ESP8266

Microcontrolador económico con Wi-Fi, ideal para proyectos de domótica, sensores conectados y control remoto.

Protoboard

Tablero para probar circuitos sin soldar.

Protoboard

Herramienta esencial para conectar componentes electrónicamente sin necesidad de soldadura.

Servomotor SG90

Servo de plástico pequeño, 180° de giro.

Servomotor SG90

Microservo común en proyectos educativos para realizar movimientos precisos en robots o puertas automáticas.

Servomotor MG90S

Microservo común con mayor precisión y fuerza que el SG90.

Servomotor MG90S

Servomotor de igual tamaño que el servomotor SG90 pero con mayor fuerza y precisión.

Servomotor MG995

Servo metálico con más fuerza y precisión.

Servomotor MG90S

Versión mejorada del SG90 con engranajes metálicos, mayor durabilidad y torque para tareas exigentes.

Relé Arduino

Interruptor eléctrico que permite encender aparatos.

Relé

Permite controlar dispositivos de alto voltaje (como bombillos o electrodomésticos) desde un microcontrolador.

Bombillo LED

Fuente de luz eficiente, controlable por relé o app.

Bombillo LED

Iluminación de bajo consumo usada en automatización del hogar. Puede ser controlado remotamente usando relés y microcontroladores.

Resistencia 200Ω

Limita el paso de corriente en un circuito.

Resistencia 200 Ohmios

Usada para proteger componentes como LEDs de sobrecorriente. Es esencial para evitar daños en circuitos electrónicos.

Arduino Uno

Placa de desarrollo básica para proyectos electrónicos.

Arduino Uno

Microcontrolador ideal para principiantes. Fácil de programar y conectar con sensores, servos y módulos electrónicos.

Puente H L298

Controlador de motores DC o paso a paso.

Puente H L298

Módulo que permite controlar la dirección y velocidad de motores, permitiendo avanzar o retroceder.

Jumpers

Cables para conectar componentes en protoboard.

Jumpers

Cables flexibles que permiten conectar pines, sensores, placas y módulos entre sí. Vienen en varias longitudes.

Enchufe Alexa

Permite encender o apagar dispositivos con la voz.

Enchufe Alexa

Dispositivo inteligente que se conecta al Wi-Fi y permite controlar electrodomésticos desde Alexa o apps móviles.

Amazon Alexa Echo Show

Altavoz inteligente con pantalla y asistente Alexa.

Amazon Echo Show

Dispositivo con pantalla táctil y el asistente Alexa integrado. Permite controlar dispositivos del hogar con voz o visualmente.

Google Assistant

Asistente virtual de Google para el hogar inteligente.

Google Assistant

Asistente de voz que permite controlar luces, enchufes y otros dispositivos mediante comandos hablados desde un móvil o altavoz.
