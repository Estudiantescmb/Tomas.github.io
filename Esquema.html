Conexión del ESP32 con Alexa: Guía Completa

¿Qué es el ESP32?

El ESP32 es un microcontrolador de bajo costo y alto rendimiento con conectividad Wi-Fi y Bluetooth, ampliamente utilizado en proyectos de automatización, robótica y domótica. Gracias a su capacidad de conectarse a internet, puede ser integrado con plataformas como Amazon Alexa para controlar dispositivos con la voz.

¿Por qué conectar el ESP32 a Alexa?

Conectar el ESP32 a Alexa permite crear una casa inteligente personalizada. Podemos encender luces, controlar motores, sensores, cerraduras o electrodomésticos usando comandos de voz. Esta integración hace que la tecnología sea accesible, cómoda y moderna, alineándose con el concepto de Internet de las Cosas (IoT).

Herramientas necesarias

  • 1 ESP32 Dev Kit
  • Cable USB (para conexión y programación)
  • Cuenta de Amazon con Alexa configurada
  • Cuenta en Sinric Pro (servicio gratuito de vinculación con Alexa)
  • Editor Arduino IDE (y librerías necesarias)
  • Componentes: LEDs, relés, servomotores, sensores, etc.
  • Conexión Wi-Fi estable

¿Qué es Sinric Pro y para qué sirve?

Sinric Pro es una plataforma que actúa como intermediaria entre Alexa y el ESP32. Permite controlar dispositivos conectados con comandos de voz sin necesidad de crear una skill personalizada. Cada dispositivo se configura en la web de Sinric, generando un Device ID y un API Key que se usan en el código del ESP32.

Esquema lógico de conexión

El flujo general de conexión es:

  • Alexa → Sinric Pro → ESP32 → Dispositivo físico

Por ejemplo, al decir: "Alexa, enciende la luz del cuarto", el comando se envía a Sinric, que a su vez lo transmite al ESP32. Este ejecuta una acción como activar un pin digital, encendiendo el relé que enciende la luz.

Pasos para la conexión

  1. Crear una cuenta en Sinric Pro y configurar un nuevo dispositivo virtual (ej. interruptor, servo, ventilador).
  2. Instalar el Arduino IDE con soporte para ESP32 (añadir la URL del board manager).
  3. Instalar las librerías necesarias: WiFi, SinricPro, ArduinoJson.
  4. Escribir o cargar el código en el ESP32 con los datos de Wi-Fi y las claves de Sinric Pro.
  5. Configurar la app Alexa para reconocer el nuevo dispositivo (detectado automáticamente).
  6. Conectar el componente físico (ej: LED al pin D2 o relé al pin D5).

Ejemplo de código básico

Este es un ejemplo resumido de cómo encender un LED con Alexa:

#include <WiFi.h>
#include <SinricPro.h>
#include <SinricProSwitch.h>

#define WIFI_SSID "TuRedWiFi"
#define WIFI_PASS "TuClaveWiFi"
#define APP_KEY   "TuAppKey"
#define APP_SECRET "TuAppSecret"
#define SWITCH_ID "TuDeviceID"

#define PIN_LED 2

bool onPowerState(const String &deviceId, bool &state) {
  digitalWrite(PIN_LED, state ? HIGH : LOW);
  return true;
}

void setup() {
  pinMode(PIN_LED, OUTPUT);
  WiFi.begin(WIFI_SSID, WIFI_PASS);
  while(WiFi.status() != WL_CONNECTED) delay(100);
  
  SinricProSwitch &mySwitch = SinricPro[SWITCH_ID];
  mySwitch.onPowerState(onPowerState);
  
  SinricPro.begin(APP_KEY, APP_SECRET);
  SinricPro.restoreDeviceStates(true);
}

void loop() {
  SinricPro.handle();
}

Aplicaciones prácticas

  • Encender/apagar luces desde la cama con comandos de voz.
  • Controlar persianas o cortinas mediante servos.
  • Activar ventiladores, bombas de agua o alarmas.
  • Crear una cerradura automática con Alexa.
  • Integración con sensores de movimiento o temperatura.
